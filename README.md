# 카우보이 파티 (Cowboy Party)

기존 카우보이 듀얼/3인 카우보이의 **2~6인 일반화 버전**. 동그랗게 둘러앉아 한 턴에 한 행동(장전·방어·빵야)을 동시에 골라 공개하고, 막지 못한 한 발에 즉시 탈락 — 마지막 1인이 승리.

## 모드
- **컴퓨터와 대결**: 나 + 봇 1~5명 (총 2~6명).
- **온라인 대전**: 호스트가 방을 만들고(최대 인원 2~6 선택) 코드로 입장, 2명 이상이면 시작. Firebase RTDB 결정적 무브로그 리플레이.

## 규칙
- 매 턴 한 가지: **장전**(+1, 최대 6) / **방어**(그 턴 나에게 오는 모든 공격 무효) / **빵야**(살아있는 아무나 1명, 총알 1발 필요·소모).
- 이번 턴 장전한 총알은 다음 턴부터 사용 → 첫 턴엔 못 쏨.
- 막지 못한 한 발 = 즉시 탈락(여러 명 동시 탈락 가능). 마지막 1인 승리, 전원 동시 탈락은 무승부.

## 스택
- Flutter (Android + Web). Firebase Realtime Database (asia-southeast1).
- 디자인: 웨스턴 황혼 · Black Han Sans/Rye 폰트 내장 · Twemoji 이미지 내장(웹 깜빡임 방지).
- 웹 배포는 `flutter build web --pwa-strategy=none` + 서비스워커 unregister.

원형 좌석/총알 궤적선 등 UI는 `lib/widgets/circular_table.dart` 참고.
