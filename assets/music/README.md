# 배경음악(BGM) 파일 위치

여기에 루프용 BGM 파일을 넣으면 코드가 자동으로 재생한다 (`lib/audio/sfx.dart`의 `Bgm`).

필요한 파일 (확장자 `.mp3`):

| 파일명 | 용도 | 어디서 재생 |
|---|---|---|
| `menu.mp3` | 메뉴·상점·랭킹·보상 화면 | `lib/screens/shell.dart` |
| `battle.mp3` | 게임(오프라인·온라인) 진행 중 | `*_game_screen.dart` |

- 파일이 없으면 그냥 무음으로 동작한다(앱은 안 깨짐). Suno 등으로 만든 뒤 이 폴더에 떨구면 됨.
- **이음새 없는 루프**여야 자연스럽다. Suno 곡은 보통 인트로/아웃트로가 있어 그대로 루프하면 끊긴다 → `tool/loop_bgm.sh` 가이드(SOUND_GUIDE.md)대로 트림/크로스페이드 가공할 것.
- 라이선스: 상업 배포 앱이므로 **저작권 프리 또는 상업 사용권 확보된 음원만** (프로젝트 LEGAL 규칙). Suno는 유료 플랜 출력물만 상업 사용 가능.
- 볼륨은 코드에서 0.55(메뉴)/0.45(전투)로 낮춰 재생하므로 원곡은 일반 레벨로 정규화해 두면 됨.
